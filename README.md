## Запуск

Установка зависимостей:

```bash
npm i
```

Первое, нужно запустить приложение:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Откройте [http://localhost:3000](http://localhost:3000) в своём браузере для просмотра.

## Тесты

Чтобы запустить тесты нужно написать команду:

```bash
npm run test
```
